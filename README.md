# Анализатор булевых функций

Консольное приложение на Java для анализа булевых функций, заданных двоичным вектором значений.

## Возможности

Ввод числа переменных N

Ввод функции в виде бинарной строки длины 2^N

Обработка ошибок ввода

Построение таблицы истинности

Вывод СДНФ и СКНФ

Удаление фиктивных переменных

Построение новой таблицы истинности
